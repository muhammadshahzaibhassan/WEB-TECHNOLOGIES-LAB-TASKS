Lab 6 - CSS Fundamentals & Box Model
-----------------------------------

Files included:
- index.html         -> Portfolio example using float + box model
- style.css          -> Stylesheet using classic float layout and box-model demo
- style-grid.css     -> Alternative modern layout using CSS Grid (optional)
- lab_report.txt     -> Simple lab report template you can fill out before submission

How to run:
1. Open index.html in your browser (double-click or use Live Server in VS Code).
2. Open DevTools (F12 or right-click -> Inspect).
3. Inspect any .project-card element and view the Box Model in the CSS/Elements panel.
4. Take screenshots of the DevTools box model (required deliverable).

Suggested deliverables for submission:
- index.html and style.css (required)
- At least one screenshot showing DevTools' Box Model for .project-card
- Short lab report (use lab_report.txt template)

Notes on grading criteria:
- Ability to conduct practical: show screenshots and a working page.
- Structure: clearly organized HTML and CSS.
- Data analysis & interpretation: explain what you changed and why (in the lab report).
- Efficiency: use responsive techniques (see style-grid.css) where appropriate.
